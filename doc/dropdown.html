Créer une dropdown ​

Objectif ​

Créer un composant dropdown ItiDropdown.vue. Une dropdown est un menu contextuel qui apparait au clic sur un élément, généralement un bouton ou une icône.

Usage ​

html
<iti-dropdown ref="dp">

	<iti-button @click="$refs.dp.toggle()">Menu</iti-button>
	
	<template #menu>
	
		<iti-dropdown-item>Item A</iti-dropdown-item>
		
		<iti-dropdown-item>Item B</iti-dropdown-item>
	
	</template>

</iti-dropdown>
<iti-dropdown ref="dp">

	<iti-button @click="$refs.dp.toggle()">Menu</iti-button>
	
	<template #menu>
	
		<iti-dropdown-item>Item A</iti-dropdown-item>
		
		<iti-dropdown-item>Item B</iti-dropdown-item>
	
	</template>

</iti-dropdown>

Comportement ​

Comme pour la modal, la dropdown doit pouvoir s'afficher et se masquer. Elle doit également posséder et exposer les deux fonctions show() et hide() plus une fonction supplémentaire toggle(). Cette dernière permet de permuter l'état de la dropdown : afficher si masquée, masquer si affichée.

La dropdown une fois ouverte doit se fermer au moindre clic dans l'application.

La méthode window.addEventListener permet de s'abonner à tous les clics de votre page.

Typescript
window.addEventListener("click", onWindowClick);
window.addEventListener("click", onWindowClick);

La méthode window.removeEventListener permet de se désabonner.

typescript
window.removeEventListener("click", onWindowClick);
window.removeEventListener("click", onWindowClick);

Veillez à bien vous abonner dans la méthode show() et à vous désabonner dans la méthode hide().

TIP

A cause de l'event propagation, il est possible que votre dropdown se masque instantanément après avoir été affichée.

Pour contourner le problème, il suffit de de s'abonner au clic dans la fonction setTimeout().

typescript
setTimeout(() => {
	window.addEventListener("click", onWindowClick);
});
setTimeout(() => {
	window.addEventListener("click", onWindowClick);
});

Slots ​

Nécessite un slot par défaut pour y insérer l'élément qui va déclencher l'affichage de la dropdown. (bouton, lien, etc.).

Un second slot menu est nécessaire pour y insérer le menu à afficher/masquer.

Template ​

La dropdown doit avoir une div principale ainsi qu'une sous div qui wrap le menu.

Style ​

css
/**
 * div principale
 */
.dropdown {
    position: relative;
}

/**
 * div contenant le menu
 */
.dropdown-menu {
    background: white;
    position: absolute;
    left: 0;
}
/**
 * div principale
 */
.dropdown {
    position: relative;
}

/**
 * div contenant le menu
 */
.dropdown-menu {
    background: white;
    position: absolute;
    left: 0;
}

Comme pour la modal, n'hésitez pas à appliquer des transitions et des transformations pour animer l'apparition et la disparition de menu.

Dropdown Item ​

Pour pouvoir ajouter des éléments au menu de la dropdown, nous allons créer un nouveau composant : ItiDropdownItem.vue.

Ce composant aura pour seul responsabilité d'encapsuler une entrée dans le menu :

html
<iti-dropdown ref="dp">

	<iti-button @click="$refs.dp.toggle()">Menu</iti-button>
	
	<template #menu>
	
		<iti-dropdown-item @click="doAction1()">
			Action n°1
		</iti-dropdown-item>

	</template>

</iti-dropdown>
<iti-dropdown ref="dp">

	<iti-button @click="$refs.dp.toggle()">Menu</iti-button>
	
	<template #menu>
	
		<iti-dropdown-item @click="doAction1()">
			Action n°1
		</iti-dropdown-item>

	</template>

</iti-dropdown>

Dropdown Item Template ​

Sachant qu'une entrée du menu est cliquable, le dropdown item devra être un bouton pour respecter la sémantique HTML.

Dropdown Item Slot ​

Comme d'habitude, le dropdown item doit posséder un slot par défaut pour pouvoir être utilisable comme désiré.

Dropdown Item Style ​

Appliquer un style minimal pour avoir des entrées de menu clair et lisible avec un border pour séparer les entrées entre elles.
